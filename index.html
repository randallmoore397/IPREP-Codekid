Stage 1 — HTML

Build the skeleton of a webpage using tags and elements

📖 Lesson 1 — What is HTML?

HTML stands for HyperText Markup Language. It is the skeleton of every webpage — it tells the browser what to show: headings, paragraphs, images, and buttons.

We use tags to mark up content. Most tags have an opening tag and a closing tag.

HTML<!-- Opening tag Content Closing tag --> <h1>Hello, World!</h1> <p>I am learning HTML today.</p> <a href="https://example.com">Click me!</a>
💡 The / in a closing tag tells the browser "this element ends here". Always close your tags!

Stage 2 — CSS

Add colours, fonts, spacing, and beautiful layouts

📖 Lesson 1 — What is CSS?

CSS stands for Cascading Style Sheets. If HTML is the skeleton, CSS is the clothes and skin — it controls colours, fonts, sizes, and spacing.

CSS SYNTAX/* selector { property: value; } */ body { background-color: #f0f4ff; font-family: Arial; } h1 { color: #e63946; font-size: 2rem; } .card { background: white; border-radius: 12px; padding: 20px; }
💡 CSS goes inside a <style> tag in the <head> of your HTML!

📖 Lesson 2 — Box Model & Flexbox

Every HTML element is a box. Padding adds space inside; margin adds space outside. Flexbox arranges elements in rows or columns easily.

BOX MODEL + FLEX.card { padding: 24px; /* space INSIDE */ margin: 20px auto; /* auto = centres horizontally */ display: flex; flex-direction: column; align-items: center; gap: 12px; }

Stage 3 — JavaScript

Make webpages interactive — clicks, alerts, DOM changes!

📖 Lesson 1 — What is JavaScript?

JavaScript is the brain of a webpage. HTML is the skeleton, CSS is the clothes — JavaScript makes everything move and respond to user actions!

JAVASCRIPT// Variables store values let name = "Alex"; let age = 15; alert("Hello!"); console.log("My name is " + name);
💡 let creates a variable — a named box that stores a value you can use later!

📖 Lesson 2 — Functions & DOM

A function is reusable code you define once and call whenever needed. The DOM is how JavaScript reads and changes HTML on the page.

FUNCTIONS + DOMfunction changeTitle() { document.getElementById("title").innerText = "Clicked! 🎉"; } // In HTML: <button onclick="changeTitle()">Click!</button>
💡 document.getElementById("id") finds the element with that id so you can change it!
